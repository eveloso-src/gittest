---------------------------------------
CICLO DE TRABAJO
---------------------------------------
git status -> ver estado

Trabajamos en los archivos

git add . -> pasar los archivos al area de staging o indice

git commit -m "mensaje"  -> confirmar los cambios en local

git push  -> subir el/los commit a la nube

Dia siguiente:

git fetch -> actualiza indices de archivos

git pull -> descarga el nuevo codigo/actualizaciones
